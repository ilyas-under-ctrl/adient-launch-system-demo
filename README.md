# LaunchOps PO Control Center

Role-based Vite prototype for project governance, purchase-order control, BOM management, launch simulation, production, packaging, warehouse transfers, customer delivery, stock, finance, and audit workflows.

## Run locally

```bash
npm install
npm run dev
```

Create a production build with `npm run build`.

## Scope

This repository is a front-end demonstration backed by in-memory sample data. Actions, documents, role switching, and credentials shown in the interface are prototype behavior only and must not be used as production authentication or persistence.

The supplied product requirements are retained in [`SRS_Launch_System_v2-bis.md`](./SRS_Launch_System_v2-bis.md).
